SPD-University - Про нас

Що собою являє SPD-University?

SPD-University - це проект, що допомагає талановитій молоді, яка хоче пов'язати своє життя з ІТ, максимально нівелювати той самий розрив між теоретичними знаннями та досвідом, необхідним для працевлаштування.

Програми SPD-University розроблені спеціалістами, що працюють в ІТ, а відтак найбільше відповідають очікуванням роботодавців. Розроблюючи програми навчання, провідні фахівці орієнтуються на те, що частина слухачів курсів буде працювати в їхній команді, а отже максимально адаптують їх під робоче середовище.

Усі програми мають "університетський" підхід. Це означає, що крім основних дисциплін зі спеціалізації у слухачів також будуть лекції та практичні заняття із суміжних дисциплін. Одним з найважливіших елементів навчання є наскрізний проект, який виконується в повноцінних командах, що складається з Back-End і Front-End розробників, а також QA. Менторами проектів виступають провідні спеціалісти з компанії SPD-Ukraine.

Команда SPD складається з найдосвідченіших спеціалістів своєї області, що готові ділитися своїм практичним досвідом. Курси безкоштовні, проте вимагають від абітурієнтів теоретичної бази.

У вас виникли питання? Ми з радістю вам допоможемо.

Де проходитимуть заняття на курсах SPD-University?

Заняття проходитимуть в нашому офісі за адресою вулиця Кривалівська (Паризької комуни) 7, 5-й поверх.

Яка тривалість курсу?

Загальна тривалість курсу 7 місяців, з яких 6 місяців - заняття з викладачами, лекції та практичні заняття, та ще 1 місяць на розробку наскрізного проекту в командах.

Чи можна суміщати роботу/навчання з навчанням на курсах в SPD-University?

Звичайно. Ми чудово розуміємо, що слухачі наших курсів можуть навчатися у ВУЗі або ж працювати, тому для зручності заняття в SPD-University розпочинаються о 18:00.

Java for Web

Java - одна з найпопулярніших мов програмування, яку викорстовують розробники програмного забезпечення на сьогоднішній день. Java використовується у веб-розробці, а також при розробці Android-додатків. Курс Java for WEB познайомить вас із мовою, основними классами і пакетами, а такжож з найпопулярнішими фреймворками для розробки веб-додатків. Даний курс вимагає високого рівня самостійної роботи. Після завершення курсу ви будете вміти писати програми на Java, а також матимите основу, що необхідно для подальшого поглиблення своїх навичок та знань у програмуванні.

Що вас чекає на курсі?

Модуль I: Java Core

Знайомство з синтаксисом та базовими пакетами Java. Основи ООП в Java. I/O. Коллекції. Concurrency.

Модуль II: Java Enterprise

Поступово розглянемо наступні інструменти з Enterprise світу: Building Tools, Java Web Basics, SQL/NoSQL, JDBC, Spring, Hibernate, еtc.

Модуль III: Проект

Front-End

Цей курс призначений для тих, хто хоче навчитися створювати сайти та динамічні веб-додатки. Задача курсу – дати зрозуміти слухачам, що таке Front-end-розробка на практиці, познайомити з основними інструментами, які використовуються програмістами у повсякденній практиці та максимально підготувати до командної роботи над реальними проектами.

Що вас чекає на курсі?

Модуль I
  • ‐ JS: від основ до глибокого занурення в ООП та функціональне програмування.
  • ‐ Продвинуті техніки HTML/CSS розробки: RWD та адаптивність, методології BEM та Atomic Design, і багато чого цікавого.
Модуль II
  • ‐ Особливість використання JS в середовищі веб-браузера.
  • ‐ Інтернет мережа очима front-end розробника, принципи побудови Single Page Application (SPA).
  • ‐ Вивчення найпопулярніших на сьогоднішній день платформ для створення веб-додатків: Angular 2 та React.

QA

Інженер забезпечення якості - це незамінна роль у сучасному проекті, без якої продукт буде мати приховані непомітні дефекти та незручності, а тому приречений на зриви дедлайнів, нескінченні скарги кінцевих користувачів та інші негаразди. Курс призначений для слухачів без професійного досвіду в сфері тестування програмного забезпечення, але із знаннями базової теорії по QA, критичним мисленням і вмінням конструктивно критикувати. Після успішного проходження цього курсу ви матимете всі необхідні теоретичні знання і практичні навички, необхідні для кандидата на Junior QA.

  • ‐ Ознайомлення з автоматизацією тестування та тестуванням безпеки.
